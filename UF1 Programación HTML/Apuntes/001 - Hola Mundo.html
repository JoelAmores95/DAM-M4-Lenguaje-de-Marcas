Atributos Principales
Lista completa aquí

  • id - Identificador
  • class - Clase para el CSS
  • style - Aplica directamente un estilo (Mejor usar CSS)
  • title - Muestra el titulo al pasar el ratón por encima
    PRUÉBAME
  • Contenteditable = "true/false" - Básicamente permite o deniega la edición
  • lang = "codigo idioma"
    es - español
    en - inglés
    fr - francés
  • dir = "LTR (Left To Right) | RTL (Right To Left)
  • accesskey = "letraDeAcceso" - Al pulsar esa tecla te lleva ahí
    Al pulsar la tecla "A" te trae aquí, o debería...
  • tabindex = "número" - Al darle al tabulador seguirá el orden establecido
    Va entre 0 y 32.767 (Este tiene asignado el numero 0)
    Mejor usarlo en cajas de texto por ejemplo, o botones.
