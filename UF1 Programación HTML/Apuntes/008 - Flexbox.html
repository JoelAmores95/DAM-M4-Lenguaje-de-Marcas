¡¡HOLA FLEXBOX, CHAO FLOAT!!

Flexbox se usa para POSICIONAR elementos en la página.
La estructura es un contenedor padre y varios elementos dentro que son los hijos.

section class = "padre" en HTML.
.padre {display: flex} en .css

Propiedades del contenedor flex (padre)

  • flex-direction: como se muestran en horizontal (X) los elementos
  • flex-wrap: Permite juntar en una única fila todos los elementos (nowrap), o en varias (wrap) o wrap-reverse
  • flex-flow: Junta las dos anteriores en una.
  • justify-content: Alinea los elementos a un lado:
    • flex-start: a la izquierda
    • flex-end: a la derecha
    • center: al centro
    • space-between: el primero y el ultimo a los bordes, el resto con espacios iguales entre ellos
    • space-around: todos tienen el mismo espacio entre ellos
  • align-items: este tiene que ver con el eje secundario (Y)
    • stretch: ocupan toda la altura
    • flex-start: arriba a la izquierda
    • flex-end: abajo a la izquierda
    • center: en todo el medio
    • baseline: las lineas base quedan a la misma altura
  • align-content: mejor mirar apuntes

Propiedades de los hijos

  • order: Asigna el valor del orden que tiene. A más bajo antes sale, puede ser negativo.
  • flex-grow: Establece el tamaño con respecto al resto
  • flex-shrink: Para hacerlo peque, si se pone 0 queda igual, no se reduce
  • flex-basis: es igual que height y width. Se pone auto o el valor en px.
  • flex: Une flex-grow, flex shrink y flex-basis
    • .hijo {flex: none|auto 0/1 0/1 auto}
  • align-self: para posicionar de forma concreta algun elemento
