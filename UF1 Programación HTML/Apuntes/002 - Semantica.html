Semántica de la web

Párrafos

Los párrafos se han explicado aquí

Negrita y cursiva

Para poner una palabra en negrita se usa la etiqueta "strong".
Las cursivas son más simples, usan la etiqueta "em" de emphasized.

Mark

Para destacar una parte de un texto se usa < mark> y luego se puede editar en css.

Enlaces

Los enlaces usan el elemento "a" con el atributo "href" + la dirección a donde llevan.
Puede ser local o externo.
Al ser visual se edita en .css

Contacto

"address" sirve para poner datos de contacto como correo electrónico.
algo así:

joel.amores95@gmail.com

Tiempo

Se usa la etiqueta "time". Sirve para la organización de los bots. Los formatos correctos están aquí.
Por ejemplo: Tengo una cita el día de San Valentín
Puedo poner fecha y hora, pero se separa usando "T" 2014.12.31 18:55:47

Citaciones

Cuando queremos hacer referencia a una publicación externa usamos < cite>texto a citar< /cite>
L'Odissea fue escrita por Homero Acabo de trobar alguna cosa sobre l'abdicació del Rei, la seva referència és ISBN 13: 978-84-617-0726-3

Para parafrasear algo con las mismas palabras exactas se usa < q>. Añade comillas automáticamente.
To be or not to be, that is the question

Cuando cogemos texto de otra fuente se usa el < blockquote>. Alguna noticia por ejemplo.

Listas

Las famosas listas. Hay dos tipos: Ordenadas, con número (< ol>) y desordenadas, con símbolos (< ul>)
Los elementos de la lista son con < li>.
Se puede elegir el número de inicio usando el atributo start="numInicio".
Se puede hacer listas de listas (Listas anidadas)
